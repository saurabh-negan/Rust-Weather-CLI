A simple command-line application built in Rust that fetches real-time weather information for any city using the OpenWeatherMap API
.
It demonstrates API integration, JSON parsing, CLI argument handling, and environment variable management in Rust.

# Get weather for Delhi
cargo run -- --city Delhi

# Get weather for London
cargo run -- --city London
